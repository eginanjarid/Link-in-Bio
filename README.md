# Eginanjar | Link-in-Bio

A premium, minimalist "Link-in-Bio" page built with Next.js, Tailwind CSS, and Framer Motion.
Designed with a "glassmorphism" aesthetic and Bento Grid layout.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Features

- **Responsive Bento Grid**: Adapts from 1 column on mobile to 2 columns on desktop.
- **Glassmorphism UI**: Frosted glass effect with blur and subtle borders.
- **Performance**: Abstract background generated with CSS/SVG (no heavy image assets).
- **Animations**: Smooth spring animations on load and hover interactions.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Customization

- Edit `src/app/page.tsx` to update links and profile information.
- Edit `src/components/Background.tsx` to change the abstract background colors.
